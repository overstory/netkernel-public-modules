
== HTTP Async Client ==

This resource encapsulates the complexity of issuing and responding to asynchronous
outbound HTTP requests.  It will initiate an asynchronous call to one of the HTTP
client endpoints (active:httpAsyncGet, for example) depending on the 'method' argument
and manage the response when it completes.
